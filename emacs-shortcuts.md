# Shortcuts emacs

## eshell

Open file: find-file <filename>

## Bookmark

Add to bookmark: ctrl + x r m
List bookmarks: ctrl + x r l

## Ruller (fill-column-indicator plugin)

Show: fci-mode

## Switch

Previous|Next buffer: C-x LEFT|RIGHT

Another window: C-x o

## Dired

Enter edit mode: C-x C-q
